2016-09-07 15:06:34

1. 新的数据结构

（1）set

  • 初始化： var s = new Set();
  • 输入参数： var s = new Set([1,2,3,4]);
  • 添加参数： s.add(1)
  • 特点： Set里面的数据不能重复，相当于使用精确对比===进行判别，’1’和1认为是两个不同的值。
    有个特例NAN和NAN在Set里面认为是相等的。两个对象始终认为是不相等的。

  • 适用场景： 数据去重[…new Set(array)] 等同于 Array.from(new Set([1,2,3,4,5]))

  • 常用方法： add(value)：添加某个值，返回Set结构本身。

    delete(value)：删除某个值，返回一个布尔值，表示删除是否成功。
has(value)：返回一个布尔值，表示该值是否为Set的成员。
clear()：清除所有成员，没有返回值。
keys()：返回键名的遍历器
values()：返回键值的遍历器
entries()：返回键值对的遍历器
forEach()：使用回调函数遍历每个成员
  • 注意事项： (1) Set的遍历顺序是插入的顺序
    (2) Set结构没有键名只有键值，因此keys()和values()方法的结果一致。
(3) set.forEach((value, key) => console.log('value', value));
